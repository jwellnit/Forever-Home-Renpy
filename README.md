# A Forever Home Ren'Py Demo

A Ren'Py remake of "A Forever Home" (https://rosevinemodding.itch.io/a-forever-home), using a reduced version of the original script. 

This version totals ~3.3k words and has 4/5 of the original endings, but excludes the explore sections from the original game: Damien is the only speaking character. No audio or visuals are used in this version; this is a pure text demo.
