Welcome to A Forever Home!

To play the game simply run the "A Forever Home.html" file in a browser and enjoy. This file must always be kept in the same directory as the included "/images" and "/music" directories, else the sound and images won't appear correctly in the game.

If you wish to edit the game's source files, you're free to! We reccomend you download the Tweego compiler (https://www.motoslave.net/tweego/). Be sure to download the SugarCube – v2.36.1 for Twine 2.1 & Tweego (https://www.motoslave.net/sugarcube/2/) for use with Tweego, as that is the story format that A Forever Home is built with. This is a good guide for working with Twine as .twee source files: https://dev.to/lazerwalker/a-modern-developer-s-workflow-for-twine-4imp
